# 🏴‍☠️ One Piece – Interactive Landing Page

An immersive **One Piece** inspired landing page built using a **single HTML file** with **inline CSS** and **JavaScript**. The project features smooth animations, interactive effects, and a modern anime-inspired interface showcasing Monkey D. Luffy and the world of One Piece.

## ✨ Features

* 🎨 Modern anime-inspired UI
* ⚡ Smooth animations and transitions
* 🖱️ Interactive mouse movement effects
* ✨ Canvas-based visual effects
* 📱 Responsive design
* 🎭 Stylish typography
* 🚀 Single HTML file (no external CSS or JS files)

## 🛠️ Built With

* HTML5
* Inline CSS3
* Vanilla JavaScript
* HTML Canvas API
* Google Fonts

## 📂 Project Structure

```
📦 OnePiece-LandingPage
 ┣ 📄 index.html
 ┗ 📄 README.md
```

## 🚀 Getting Started

1. Clone this repository.

```bash
git clone https://github.com/your-username/onepiece-landing-page.git
```

2. Open `index.html` in your browser.

That's it! No installation or dependencies required.
